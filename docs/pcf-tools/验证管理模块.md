# 验证管理模块

# 功能模块介绍

该模块是AICP平台提供的第三方验证功能，帮助用户基于已经产生的碳足迹结果形成一个第三方验证记录。

该模块支持用户直接引用一项已经产生的、针对特定产品批次的碳足迹结果，并上传与该结果相关的支持性材料（文档、媒体文件等），以形成一个待验证项。

该模块支持用户将待验证项授权给第三方验证方用户查看并验证碳足迹结果。例如，用户可以将他的某一项产品碳足迹结果连同《盘查报告》以及相关支持性材料授权给第三方认证机构进行查验。获取授权的第三方认证机构有权在授权范围内查验用户的待验证项内容，但无权更改待验证项本身的内容。

获取授权的第三方认证机构可以在完成在线数据、材料、文档的查验，以及相匹配的线下核查工作后，在待验证项中变更验证状态，并上传相应的《核查报告》或其他适用的验证文档。

# 使用指南

## 新建验证记录（待验证项）

点击界面右上角“新建验证记录“：

![截屏2023-08-16 17.34.09.png](/img/2023-08-16_17.34.09.png)

用户在发起验证时指定需要验证的碳足迹批次（要求已经存在计算结果）以及授权的第三方验证人（如实施碳足迹项目中对应的第三方认证机构用户）。

在发起验证时，用户还需上传一个文件夹，该文件夹包含所有用于支持待验证项的材料、文档、媒体文件等。请确保所有支持文件都被存放在一个专用的文件夹中。

![截屏2023-08-16 17.41.08.png](/img/2023-08-16_17.41.08.png)

如上图所示，用户成功发起一个待验证项后，该验证记录会显示在“验证管理列表”中。

用户可以在该待验证项被第三方验证人所验证之前，查看、修改完善待验证项的内容。当该待验证项被第三方验证人所验证之后，用户将无权修改验证记录的内容。

## 第三方验证人完成验证

如果您是第三方验证人用户，你可以进入该工具界面查看所有对您开放授权的待验证项。

正在对您开放授权的待验证项会自动显示在该工具界面。您可以查看该待验证项内的所有内容，包括待验证项的信息以及附加的文件夹。

如果您已经完成所有的验证工作，请在该待验证项界面中上传相应的验证文件（包括但不限于核查报告、审计报告、各类意见书等）并置验证状态为“已验证”。需要注意，一旦验证状态被置为“已验证”，被验证的用户将无法修改验证项内的任何内容（包括验证记录的信息以及附带文件夹的内容）。
